Why Veil

Built different.
Actually.

Veil takes everything LiquidBounce Nextgen offers and improves on it. More control, better performance, cleaner experience.

⚔

Blatant Combat Suite

Full KillAura with multi-target, rotation modes, and range control. AimBot with smooth aim assist, FOV limiting, and target priority. Reach extension, Velocity reduction, CriticalHit, AutoWeapon switching, and MaceKill for modern combat. Everything tunable per-module with fine-grained settings.

🚀

Movement Freedom

Creative-style Flight with speed control. Vanilla-breaking Speed with multiple bypass modes. NoFall, Jesus (water walking), BHop, LongJump, and Parkour automation. Phase through blocks, NoWeb to ignore cobwebs, and full Sprint control with collision ignore.

👁

ESP & Rendering

Player ESP with distance, health, and armor display. Tracers, Chams for through-wall player outlines, StorageESP for chests and shulkers, ItemESP, and BreadCrumbs trail. Fullbright, custom Nametags, Minimap with entity radar, and a fully customizable drag-and-drop HUD.

🌍

World Control

Scaffold for automatic bridging. Nuker to instantly break surrounding blocks. BedNuker for crystal PvP setups. AutoMine, FastBreak, ChestStealer, FastPlace, AutoTool, and TileSpoofer. Everything you need to control the world around you.

💀

Exploit Modules

Blink to freeze and release your position packet. Backtrack to delay enemy positions. AntiKick to prevent server timeouts. PacketFly for movement-based flight. PingSpoof, KickDesync, AntiBot, and Disabler. Push every server limit.

🎨

Custom HUD & UI

LiquidBounce Nextgen's drag-and-drop HUD editor lets you place, resize, and customize every element. Module list, player info, armor display, inventory stats, minimap — all fully configurable. Multiple built-in themes with full color control.

⚡

Performance

Built on Fabric with performance-first architecture. Veil adds zero overhead beyond what the modules need. High FPS, low latency, no stutter. Runs clean alongside other Fabric performance mods like Sodium.

🔄

Regular Updates

Veil tracks every LiquidBounce Nextgen release. New Minecraft versions, new modules, and new bypass improvements are pushed regularly. Join the Discord to always get the latest build the moment it drops.

🧠

Script & Plugin API

Inherited from LiquidBounce Nextgen, Veil supports a JavaScript plugin API. Write your own modules in a few lines of code, share them with others, or load community scripts. Full access to the client internals — if a module doesn't exist yet, build it yourself.
